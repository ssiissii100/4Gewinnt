4 Gewinnt – Regelzusammenfassung

Spielvorbereitung

  • 4 Gewinnt wird von zwei Spielern gespielt.
  • Jeder Spieler wählt eine Farbe.
  • Das Spielbrett besteht aus sieben Spalten und sechs Reihen.

Spielablauf

  • Die Spieler ziehen abwechselnd.
  • Jeder Spieler lässt einen seiner Spielsteine in eine der Spalten des Gitters fallen.
  • Der Stein fällt immer so weit wie möglich nach unten.

Ziel des Spiels

  • Ziel ist es, als erster Spieler eine ununterbrochene Linie von vier eigenen Steinen zu bilden.
  • Diese Linie kann waagerecht, senkrecht oder diagonal verlaufen.

Spielende

  • Das Spiel endet, sobald ein Spieler vier Steine in einer Linie hat.
  • Sollte das Spielfeld voll sein und kein Spieler hat vier Steine in einer Linie, endet das Spiel unentschieden.
